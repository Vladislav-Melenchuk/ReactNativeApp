## Демонстрація роботи

### Авторизація

Демонстрація роботи екранів входу та реєстрації користувача. Реалізовано сучасний інтерфейс, перемикання між режимами авторизації та базову валідацію введених даних.

![Авторизація](part1.gif)

---

### Анімації

Демонстрація реалізованих анімацій із використанням переміщення, масштабування, повороту та зміни прозорості. Передбачено вибір пресетів і запуск анімації.

![Анімації](part2.gif)

---

### Курси валют

Демонстрація сторінки курсів валют НБУ. Реалізовано вибір дати, завантаження актуальних курсів, відображення дати оновлення та використання прототипного методу `Date` для формування запиту.

![Курси валют](part3.gif)
